# Username Generator

A simple Python program that creates a username using a user's first name, last name, and birth year.

## Features

* Generates usernames automatically
* Validates birth year input
* Converts names to lowercase
* Displays username length
* Shows the username in uppercase format
* Allows multiple username generations in one run

## Requirements

* Python 3.x

## Example

Enter your first name : Aayush
Enter your last name : Shinde
Enter your birth year : 2012

Username : aayush_shinde_2012
Length   : 18
Uppercase: AAYUSH_SHINDE_2012
